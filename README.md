<sub>User script mod for the incremental game [Synergism](https://synergism.cc/). Also available on [Steam](https://store.steampowered.com/app/3552310/Synergism/).</sub>
_______
![mod screenshot](https://github.com/AFKoncore/synergism-QoL/blob/main/demo.png?raw=true)
### Adds some QoL buttons to make Singularities less tedious
<sub>Mouse over any of the mod's button for a couple seconds to get a tooltip explanation in game.</sub>
- Open all cubes in one click without changing tab (8 clicks + mouse travel saved every time!)
- Buy all early singularity 3D-WowCubes with a single button.
- Ant sacrifice + 'BUY ALL' Talismans shards and fragments in one click without changing tab.

### [Install](https://raw.githubusercontent.com/AFKoncore/synergism-QoL/refs/heads/main/synQoL.user.js)
<sub>Click the link above, you need a browser extension that automatically loads user scripts, such as [Violentmonkey](https://violentmonkey.github.io/get-it/).</sub>
_______
Also check out [Hypersynergism](https://github.com/Ferlieloi/synergism-hypersynergy) for a handy bar that display the current state of various automation toggles that are otherwise hidden away across many tabs and sub-menus.   
<sub>Enable it by clicking the mod button top left, then Settings > QOL-Buttons > Automation quick bar.</sub>
